# Kendo.Mvc.UI.Fluent.AutoCompleteBuilder

## Methods

### HighlightFirst(System.Boolean)
Use it to enable highlighting of first matched item.

#### Example
    <%= Html.Kendo().AutoComplete()
        .Name("AutoComplete")
        .HighlightFirst(true)
        %>
