# Kendo.Mvc.UI.Fluent.DropDownListBuilder

## Methods

### SelectedIndex(System.Int32)
Use it to set selected item index

#### Example
    <%= Html.Kendo().DropDownList()
        .Name("DropDownList")
        .SelectedIndex(0);
        %>

#### Parameters

##### index `System.Int32`
Item index.
