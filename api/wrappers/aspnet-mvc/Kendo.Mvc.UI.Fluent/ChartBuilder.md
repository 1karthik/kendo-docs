# Kendo.Mvc.UI.Fluent.ChartBuilder

## Methods

### Transitions(System.Boolean)
Enables or disabled animated transitions on initial load and refresh.

#### Example
    <%= Html.Kendo().Chart()
        .Name("Chart")
        .Transitions(false)
        %>

#### Parameters

##### transitions `System.Boolean`

            A value indicating if transition animations should be played.
