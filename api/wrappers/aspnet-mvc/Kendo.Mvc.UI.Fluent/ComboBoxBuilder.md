# Kendo.Mvc.UI.Fluent.ComboBoxBuilder

## Methods

### SelectedIndex(System.Int32)
Use it to set selected item index

#### Example
    <%= Html.Kendo().ComboBox()
        .Name("ComboBox")
        .SelectedIndex(0);
        %>

#### Parameters

##### index `System.Int32`
Item index.
