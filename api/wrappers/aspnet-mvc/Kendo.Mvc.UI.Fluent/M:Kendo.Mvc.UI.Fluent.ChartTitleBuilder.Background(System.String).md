# Kendo.Mvc.UI.Fluent.ChartTitleBuilder

## Methods

### Background(System.String)
Sets the title background color

#### Example
    <% Html.Kendo().Chart()
        .Name("Chart")
        .Title(title => title.Background("red"))
        .Render();
        %>

#### Parameters

##### background `System.String`
The background color.
