# Kendo.Mvc.UI.Fluent.ChartBuilder

## Methods

### Theme(System.String)
Sets the theme of the chart.

#### Example
    <%= Html.Kendo().Chart()
        .Name("Chart")
        .Theme("Telerik")
        %>

#### Parameters

##### theme `System.String`
The Chart theme.
