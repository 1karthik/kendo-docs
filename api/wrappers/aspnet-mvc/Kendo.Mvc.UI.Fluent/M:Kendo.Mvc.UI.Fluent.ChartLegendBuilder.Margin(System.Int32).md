# Kendo.Mvc.UI.Fluent.ChartLegendBuilder

## Methods

### Margin(System.Int32)
Sets the legend margin

#### Example
    <% Html.Kendo().Chart()
        .Name("Chart")
        .Legend(legend => legend.Margin(20))
        .Render();
        %>

#### Parameters

##### margin `System.Int32`
The legend margin.
