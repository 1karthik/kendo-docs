# Kendo.Mvc.UI.Fluent.ChartBuilder

## Methods

### SeriesColors(System.String[])
Sets the series colors.

#### Example
    <%= Html.Kendo().Chart()
        .Name("Chart")
        .SeriesColors("#f00", "#0f0", "#00f")
        %>

#### Parameters

##### colors `System.String[]`
The series colors.
