# Kendo.Mvc.UI.Fluent.ChartLegendBuilder

## Methods

### Padding(System.Int32)
Sets the legend padding

#### Example
    <% Html.Kendo().Chart()
        .Name("Chart")
        .Legend(legend => legend.Padding(20))
        .Render();
        %>

#### Parameters

##### padding `System.Int32`
The legend padding.
