# Kendo.Mvc.UI.Fluent.ChartBuilder

## Methods

### Legend(System.Boolean)
Sets the legend visibility.

#### Example
    <%= Html.Kendo().Chart()
        .Name("Chart")
        .Legend(false)
        %>

#### Parameters

##### visible `System.Boolean`
A value indicating whether to show the legend.
