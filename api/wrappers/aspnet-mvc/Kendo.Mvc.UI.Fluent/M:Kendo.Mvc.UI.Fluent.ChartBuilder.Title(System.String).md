# Kendo.Mvc.UI.Fluent.ChartBuilder

## Methods

### Title(System.String)
Sets the title of the chart.

#### Example
    <%= Html.Kendo().Chart()
        .Name("Chart")
        .Title("Yearly sales")
        %>

#### Parameters

##### title `System.String`
The Chart title.
