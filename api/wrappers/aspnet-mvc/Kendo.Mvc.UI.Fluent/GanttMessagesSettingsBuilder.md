# Kendo.Mvc.UI.Fluent.GanttMessagesSettingsBuilder
Defines the fluent API for configuring the GanttMessagesSettings settings.




## Methods


### Views(System.Action\<Kendo.Mvc.UI.Fluent.GanttMessagesViewsSettingsBuilder\>)
