# Kendo.Mvc.UI.Fluent.ChartTitleBuilder

## Methods

### Padding(System.Int32)
Sets the title padding

#### Example
    <% Html.Kendo().Chart()
        .Name("Chart")
        .Title(title => title.Padding(20))
        .Render();
        %>

#### Parameters

##### padding `System.Int32`
The title padding.
