# Kendo.Mvc.UI.Fluent.GridToolBarSaveCommandBuilder

### 1.SaveText(System.String)
Sets the text displayed by the "save changes" button. If not set a default value is used.

#### Parameters

##### text `System.String`
The text which should be displayed
