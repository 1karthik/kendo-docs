# \<kendo:diagram-shape-rotation\>

The function that positions the connector.

#### Example
    <kendo:diagram-shape>
        <kendo:diagram-shape-rotation></kendo:diagram-shape-rotation>
    </kendo:diagram-shape>

## Configuration Attributes

### angle `float`

The rotation angle.

#### Example
    <kendo:diagram-shape-rotation angle="angle">
    </kendo:diagram-shape-rotation>
