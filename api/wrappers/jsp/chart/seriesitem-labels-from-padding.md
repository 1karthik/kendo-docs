# \<kendo:chart-seriesItem-labels-from-padding\>

The padding of the from labels. A numeric value will set all paddings.

#### Example
    <kendo:chart-seriesItem-labels-from>
        <kendo:chart-seriesItem-labels-from-padding></kendo:chart-seriesItem-labels-from-padding>
    </kendo:chart-seriesItem-labels-from>

## Configuration Attributes

### bottom `float`

The bottom padding of the from labels.

#### Example
    <kendo:chart-seriesItem-labels-from-padding bottom="bottom">
    </kendo:chart-seriesItem-labels-from-padding>

### left `float`

The left padding of the from labels.

#### Example
    <kendo:chart-seriesItem-labels-from-padding left="left">
    </kendo:chart-seriesItem-labels-from-padding>

### right `float`

The right padding of the from labels.

#### Example
    <kendo:chart-seriesItem-labels-from-padding right="right">
    </kendo:chart-seriesItem-labels-from-padding>

### top `float`

The top padding of the from labels.

#### Example
    <kendo:chart-seriesItem-labels-from-padding top="top">
    </kendo:chart-seriesItem-labels-from-padding>
