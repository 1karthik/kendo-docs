# \<kendo:stockChart-chartArea\>
A JSP tag representing Kendo ChartArea.

#### Example
    <kendo:stockChart>
        <kendo:stockChart-chartArea></kendo:stockChart-chartArea>
    </kendo:stockChart>


## Configuration Attributes


### background `String`

The background color of the chart area.

#### Example
    <kendo:stockChart background="background">
    </kendo:stockChart>



### opacity `float`

The background opacity of the chart area.

#### Example
    <kendo:stockChart opacity="opacity">
    </kendo:stockChart>



### height `float`

The height of the chart area.

#### Example
    <kendo:stockChart height="height">
    </kendo:stockChart>



### margin `float`

The margin of the chart area.

#### Example
    <kendo:stockChart margin="margin">
    </kendo:stockChart>



### width `float`

The width of the chart area.

#### Example
    <kendo:stockChart width="width">
    </kendo:stockChart>
