# \<kendo:stockChart-seriesItem-labels-border\>
A JSP tag representing Kendo Border.

#### Example
    <kendo:stockChart-seriesItem-labels>
        <kendo:stockChart-seriesItem-labels-border></kendo:stockChart-seriesItem-labels-border>
    </kendo:stockChart-seriesItem-labels>


## Configuration Attributes


### color `String`

The color of the border.

#### Example
    <kendo:stockChart color="color">
    </kendo:stockChart>



### dashType `String`

The dash type of the border.

#### Example
    <kendo:stockChart dashType="dashType">
    </kendo:stockChart>



### width `float`

The width of the border.

#### Example
    <kendo:stockChart width="width">
    </kendo:stockChart>
