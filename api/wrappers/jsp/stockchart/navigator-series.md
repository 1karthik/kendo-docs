# \<kendo:stockChart-navigator-series\>
A JSP tag representing Kendo Series.

#### Example
    <kendo:stockChart-navigator>
        <kendo:stockChart-navigator-series></kendo:stockChart-navigator-series>
    </kendo:stockChart-navigator>
