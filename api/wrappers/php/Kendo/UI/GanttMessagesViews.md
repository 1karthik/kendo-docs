# \Kendo\UI\GanttMessagesViews

A PHP class representing the views setting of GanttMessages.


## Methods

### day
The text similar to "Day" displayed as gantt "day" view title.

#### Returns
`\Kendo\UI\GanttMessagesViews`

#### Parameters

##### $value `string`



#### Example 
    <?php
    $views = new \Kendo\UI\GanttMessagesViews();
