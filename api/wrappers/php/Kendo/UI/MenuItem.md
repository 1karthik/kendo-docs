# \Kendo\UI\MenuItem

A PHP class representing the item setting of MenuItems.


## Methods

### enabled
Specifies whether the item is initially enabled

#### Returns
`\Kendo\UI\MenuItem`

#### Parameters

##### $value `boolean`



#### Example 
    $item = new \Kendo\UI\MenuItem();
    $item->enabled(true);

### imageUrl
Specifies the URL of the image displayed by the item

#### Returns
`\Kendo\UI\MenuItem`

#### Parameters

##### $value `string`



#### Example 
    $item = new \Kendo\UI\MenuItem();
    $item->imageUrl('value');

### selected
Specifies whether the item is initially selected

#### Returns
`\Kendo\UI\MenuItem`

#### Parameters

##### $value `boolean`



#### Example 
    $item = new \Kendo\UI\MenuItem();
    $item->selected(true);
