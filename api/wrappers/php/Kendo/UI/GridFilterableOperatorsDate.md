# \Kendo\UI\GridFilterableOperatorsDate

A PHP class representing the date setting of GridFilterableOperators.


## Methods

Enable/Disable and set the text of the "Is equal to" filter option for columns with date values.

#### Returns
`\Kendo\UI\GridFilterableOperatorsDate`

#### Parameters

##### $value `string`



#### Example
