# \Kendo\UI\GridSortable

A PHP class representing the sortable setting of Grid.


## Methods

### allowUnsort
Defines whether column can have unsorted state.
#### Parameters

##### $value `boolean`



#### Example 
    $sortable = new \Kendo\UI\GridSortable();
    $sortable->allowUnsort(true);
