# \Kendo\UI\WindowAnimationOpen

A PHP class representing the open setting of WindowAnimation.


## Methods

### duration
Difines the animation duration.
#### Parameters

##### $value `float`



#### Example 
    $open = new \Kendo\UI\WindowAnimationOpen();
    $open->duration(1);

### effects
Effect to be used for opening of the popup.
#### Parameters

##### $value `string`



#### Example 
    $open = new \Kendo\UI\WindowAnimationOpen();
    $open->effects('value');
