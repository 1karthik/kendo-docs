# \Kendo\UI\GridFilterableMessages

A PHP class representing the messages setting of GridFilterable.


## Methods

### _and
Set the text of the "and" option from logic drop down list.
#### Parameters

##### $value `string`



#### Example 
    $messages = new \Kendo\UI\GridFilterableMessages();
    $messages->_and('value');

### clear

##### $value `string`
